# Volcano

Volcano is an OCaml-based compiler project ported from the original Java-based VC (a Variant of C) compiler.

## Getting Started
**Prerequisites**:
- OCaml 5.0+
- opam 2.0+
- dune 3.0+

## Usage
To compile, run
```
dune build
```
